How GA Cash 3 (Pick 3) Works

You pick any 3-digit number from 000 to 999. Each evening the Georgia Lottery draws one winning 3-digit number. There are 1,000 equally likely outcomes per draw.

Bet Types & Payouts (per $1 ticket)

  • Straight — exact order match. Pays $500. Odds: 1 in 1,000.
  • Box (all-different digits) — any order, 6 combinations. Pays $80. Odds: 6 in 1,000 (1 in ~167).
  • Box (one-pair digits) — any order, 3 combinations. Pays $160. Odds: 3 in 1,000 (1 in ~333).

The House Edge — Why the Game Works Against You

For every dollar wagered, the Georgia Lottery pays back only about 48–50 cents on average. That means the house keeps 50–52 cents of every dollar — every single draw.

  • Straight: expected return = $0.50 per $1 → 50% house edge
  • Box (6-way): expected return = $0.48 per $1 → 52% house edge
  • Box (3-way): expected return = $0.48 per $1 → 52% house edge

For comparison, casino slot machines typically run a 5–10% house edge and roulette about 5.3%. Pick 3 has roughly 10× the house edge of most casino games. No tracking system, hot/cold analysis, or pattern recognition can change these odds — each draw is statistically independent.
